AI-Driven Learning Path Dynamic Cycle

Stage 1: Plan

Input: Goals & Ability Assessment
↓
Process: AI Path Planning Engine
↓
Output: Personalized Learning Map

The system generates an optimal learning path as an "ideal blueprint" based on the student's goals and current abilities.

Stage 2: Monitor

Core Monitoring Logic

Cyclic comparison: Actual Progress vs Planned Path

When significant deviation is detected → Trigger risk assessment

The system continuously tracks learning data in the background, comparing the student's actual progress with the planned blueprint in real-time.

Stage 3: Dynamic Adjustment

Event: Progress Assessment
↓
Progress Lag
↓
⚠️ Risk Warning & Supplementary Learning
Progress Ahead
↓
📚 Learning Extension
⚡ Linear Acceleration

The system provides intelligent adjustment suggestions for both positive and negative scenarios based on monitoring results.

Stage 4: Adapt

User Decision: Accept Suggestion / Adjust Goals
↓
System Action: Update Learning Path
↓
Complete Loop:
Return to "Monitor" Stage

The system dynamically adjusts the learning path based on user decisions, completes the loop and starts a new round of monitoring.
